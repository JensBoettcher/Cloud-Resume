Über Mich

Hey, schön, dass du den Weg auf meine Seite gefunden hast! Ich bin Jens und beschäftige mich seit über 20 Jahren mit Computertechnologien. Angefangen mit dem Computer der Familie auf dem noch Windows 98 lief, über den selbst zusammengebauten Computer habe ich den rasenden Fortschritt an Rechenleistung und Innovationen hautnah mit erlebt.

Vor etwas mehr als zwei Jahren habe ich die Azure Cloud für mich entdeckt. Seit über 10 Jahren habe kein Laufwerk außer eine Festplatte in meinen PC. Das Internet machte es möglich. Mich begeistert der Gedanke durch die Cloudtechnologie, Rechenleistung auf Knopfdruck zu erhalten. Die weiteren Entwicklungen in der Cloud Welt möchte ich auf professioneller Ebene miterleben. Deshalb fing ich an Bash Scripting, Netzwerkgrundlagen und diverse Cloudservices in meiner Freizeit zu lernen.

Diese Website ist Teil der Azure Cloud Resume Challenge. Durch diese Challenge werden die Fähigkeiten und das Wissen über CosmosDB, Azure Function, Azure Blob Storage und Azure Custom Domain Service vertieft. Zusätzlich erstelle ich ein Azure CLI Script, um alle benötigten Ressourcen automatisiert bereit zustellen.

Derzeit arbeite ich intensiv daran, das AZ-104 Zertifikat zu erlangen, das voraussichtlich im Mai abgeschlossen sein wird. Dieses Zertifikat wird nicht nur meine Fähigkeiten in Azure bestätigen, sondern mir auch ermöglichen, Projekte auf professioneller Ebene umzusetzen.

Auf meiner Website will ich meine Projekte auf dem Weg zum Microsoft Certified: Azure Administrator Associate und darüber hinaus teilen.
